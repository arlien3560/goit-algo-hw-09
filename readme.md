### Результати бенчмарку на локальній машині

| Сума   | Greedy    | DP       | DP повільніший у |
|--------|-----------|----------|------------------|
| 113    | 0.0009 с  | 0.0001 с | —                |
| 1000   | 0.0002 с  | 0.0006 с | 2.5              |
| 10000  | 0.0002 с  | 0.005 с  | 24               |
| 50000  | 0.0002 с  | 0.035 с  | 167              |

**Висновок:** При зростанні суми DP сповільнюється лінійно, Greedy залишається константним. 
Для великих сум (>10000) різниця стає критичною.
